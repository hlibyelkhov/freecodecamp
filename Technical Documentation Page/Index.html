HTML

HTML stands for HyperText Markup Language.

It is the standard markup language for creating web pages and web applications.

HTML uses various elements and attributes to structure and present content on the web.

<p>This is a paragraph element</p> <a href="#"></a>

CSS

CSS stands for Cascading Style Sheets.

It is used to control the layout and appearance of HTML elements on a webpage.

CSS allows you to apply styles such as colors, fonts, and spacing to your web pages.

body { background-color: #f9f9f9; } h1 { color: #007bff; }

  • Properties
  • Values
  • Layout
JavaScript

JavaScript is a programming language for the web.

It is used to add interactivity and dynamic content to web pages.

With JavaScript, you can respond to user actions, manipulate the DOM, and much more.

const greeting = "Hello, World!"; document.getElementById("demo").innerHTML = greeting;

  • Functions

Responsive Design

Responsive design ensures websites look good on all devices.

By using media queries and flexible layouts, web pages can adapt to different screen sizes.

This makes your content accessible and user-friendly across various devices.

@media screen and (max-width: 768px) { /* CSS styles for mobile devices */ } @media screen and (min-width: 1200px) { /* CSS styles for large screens */ }
  • Media Queries
  • Viewport
  • Breakpoints
  • Fluid Layouts
